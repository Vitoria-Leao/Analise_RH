# Analise_RH
Este repositório vai armazenar scripts em Python e SQL, um dashboard Power BI e arquivos relacionados, usados para análise de dados de recursos humanos.

## Objetivo

Analisar fatores associados ao turnover e identificar oportunidades de retenção.

## Ferramentas
- SQL;
- Python;
- Pandas;
- Power BI;
- DAX.

## Dataset
IBM HR Analytics Employee Attrition Dataset.

## KPIs
- Turnover %;
- Salário Médio;
- Tempo Médio de Empresa;
- Total de Funcionários.

## Principais Insights
- O turnover geral foi de 16%;
- O departamento de vendas apresentou a maior rotatividade;
- Funcionários desligados possuem salário médio inferior aos ativos;
- Os desligamentos concentram-se nos primeiros anos.

## Recomendações
- Reforçar programas de onboarding;
- Revisar políticas de remuneração em áreas com alta rotatividade.
